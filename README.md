# nerdfonts
Script to install nerdfonts, only the most popular, here the list:

	Agave
	CascaidaCode
	CodeNewRoman
	DejaVuSansMono
	DroidSansMono
	FiraCode
	FiraMono
	Hack
	Iosevka
	JetBrainsMono
	LiberationMono
	Meslo
	Monoid
	NerdFontsSymbolsOnly
	Ubuntu
	UbuntuMono
	VictorMono

Commands

```bash
mkdir build
cd build
git clone git@github.com:CarlosMolinesPastor/nerdfonts.git
cd nerdfonts
chmod +x nerdinstall.sh
./nerdinstall.sh
```
